# App Store Screenshots

## Required Screenshot Sizes

### iPhone Screenshots
1. **iPhone 6.7"** (1290 x 2796 pixels) - iPhone 15 Pro Max, 14 Pro Max, 13 Pro Max, 12 Pro Max
2. **iPhone 6.5"** (1242 x 2688 pixels) - iPhone 11 Pro Max, XS Max  
3. **iPhone 5.5"** (1242 x 2208 pixels) - iPhone 8 Plus, 7 Plus, 6s Plus, 6 Plus

### iPad Screenshots  
1. **iPad Pro 6th Gen 12.9"** (2048 x 2732 pixels)
2. **iPad Pro 2nd Gen 12.9"** (2048 x 2732 pixels)

## Screenshot Content Strategy

### Screenshot 1: Main Gameplay
- Show paper airplane in flight through Sunny Meadows environment
- Include UI elements showing score and controls
- Capture dynamic moment with particle effects

### Screenshot 2: Environment Showcase
- Split screen or collage showing multiple environments
- Highlight visual diversity (Alpine Heights, Coastal Breeze, Urban Skyline)
- Emphasize beautiful graphics and variety

### Screenshot 3: Customization
- Show airplane customization screen in Hangar
- Display different paper airplane designs and fold types
- Highlight personalization options

### Screenshot 4: Game Modes
- Show game mode selection screen
- Highlight Challenge, Daily Run, Weekly Special options
- Include Game Center integration elements

### Screenshot 5: Achievements & Progress
- Display Game Center leaderboards and achievements
- Show progression system with XP and unlocks
- Emphasize competitive and social aspects

## Screenshot Guidelines

### Technical Requirements
- Use actual in-game screenshots (no mockups)
- Ensure high quality and sharp images
- Test on actual devices for accuracy
- Include status bar or hide it consistently

### Visual Guidelines
- Use portrait orientation for iPhone
- Use landscape orientation for iPad where appropriate
- Ensure good contrast and readability
- Show engaging, dynamic gameplay moments

### Text Overlays (Optional)
- Keep text minimal and readable
- Use consistent font and styling
- Highlight key features without cluttering
- Ensure text works on all device sizes

## File Naming Convention
- `iPhone_67_Screenshot_1_Gameplay.png`
- `iPhone_65_Screenshot_2_Environments.png`
- `iPad_Pro_Screenshot_1_Gameplay.png`
- etc.

## Localization
- Create screenshots for primary markets if localized
- Ensure UI text is appropriate for target languages
- Consider cultural preferences in visual presentation
