# Product Overview

Tiny Pilots is a premium iOS game featuring paper airplane flight simulation built with SpriteKit. Players control paper airplanes through various environments using tilt-based controls.

## Core Features
- Realistic paper airplane physics simulation
- Multiple environments (Sunny Meadows, Alpine Heights, Coastal Breeze, Urban Skyline, Desert Canyon)
- Customizable paper airplane designs and fold types
- Game modes: Tutorial, Free Play, Challenge, Daily Run, Weekly Special
- Game Center integration with leaderboards and achievements
- Progression system with XP, levels, and unlockable content

## Target Platform
- iOS 16.0+ (deployment target iOS 18.0+)
- iPhone 8 or newer / iPad 5th generation or newer
- Supports all orientations

## Performance Goals
- 60 FPS (120 FPS on ProMotion displays)
- Initial load under 3 seconds
- Level loads under 1 second
